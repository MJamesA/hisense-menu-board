# Hisense Digital Menu Board

Full-screen auto-rotating digital signage for shop TVs.

- **`index.html`** — the menu board. Open full-screen on any TV browser (no login needed).
- **`data.json`** — live product feed, mirrored from the internal Price Tags Google Sheet by an Apps Script timer.

Live page (GitHub Pages): https://mjamesa.github.io/hisense-menu-board/

Data is read-only and contains only public signage fields (category, name, model, prices, features).
